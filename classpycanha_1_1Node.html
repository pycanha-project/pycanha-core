◆ _parent_pointer

std::weak_ptr<Nodes> pycanha::Node::_parent_pointer
private

Stores a weak pointer to the parent Nodes instance where the node belongs. The weak pointer is created from a shared pointer in TNs. When the TNs instance is destroyed, so is the shared pointer. And TN knows that the parent instance is not accesible any more
